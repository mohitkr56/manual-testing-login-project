# Manual Testing Project — Login Module

## 📌 Project Overview
This project demonstrates manual testing on a demo web application login module.  
It includes test scenarios, test cases, bug reporting, and test summary documentation.

## 🎯 Objective
To verify login functionality using manual testing techniques and identify defects.

## 🧪 Scope of Testing
• Valid login  
• Invalid login  
• Empty fields validation  
• Case sensitivity check  
• Special character testing  
• Multiple login attempts  

## 📂 Project Files
• TestCases.xlsx — Contains detailed test cases  
• BugReport.docx — Contains reported defects  
• TestSummary.docx — Contains execution summary  

## 🛠 Tools Used
• MS Excel  
• MS Word  
• Google Chrome  

## ✅ Outcome
The login module worked correctly for most scenarios. Minor issues were identified and documented.
